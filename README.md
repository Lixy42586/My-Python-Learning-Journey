# 成绩计算程序
本程序是一个用于中小学考试成绩计算的python程序，允许教师调整试卷的满分和题目数量，通过每题扣分计算学生成绩，并支持连续输入多个学生成绩。
## 程序功能
本程序能够支持自定义试卷总分和题目数量，连续输入多个学生成绩，对于简单的输入错误（扣分分数输入超过20分、数据格式错误等）进行检测和重新输入，自动记录所有学生成绩，并在退出时显示全部学生成绩汇总。

## 使用方法

### 环境要求

使用Python 3.6 或更高版本，下载程序代码并运行。

### 运行流程

#### 1.首次设置

输入试卷的总题数（记分区域总数），输入试卷的卷面总分。

#### 2.输入学生成绩

输入学生姓名或学号按题号顺序依次输入每题的扣分值，程序会自动计算并显示最终成绩。

#### 3.继续或退出

输入"是"退出程序，退出时会显示所有学生的成绩汇总。

#### 4.错误检测及重新输入

数据类型或输入错误时会直接提示重新输入，再次输入即可。

分数输入错误后直接继续输入下一题扣分数据即可，所有题号输入完成后会出现重新输入界面，直接重新输入即可。

处理单个学生的成绩输入，包含错误检测机制：

while True:    try:        name = input("请输入姓名/学号：")        error = 0        *# 输入学生姓名学号后，开始按题号依次输入扣分*        for i in Total_Part:            kf = float(input("请输入第" + str(i) + "部分扣分(正值，单次不大于20分)："))            *# 卷面单题扣分一般不大于20分，可以简单排查出输入错误*            if kf >= 20:                error = error + 1                continue            All = All - kf

if error >= 1:    elist = list(range(1, error + 1))    for j in elist:        kf = float(input("请修正第" + str(j) + "次错误输入："))        All = All - kf

显示成绩、记录学生信息，并重置分数为初始值：

print("总成绩是：" + str(All)) new_stu = (name, All) All_stu = All_stu + new_stu All=origin_All

tc = input("你是否想退出：") if tc == "是":    print(All_stu)    break

处理类型错误和数值错误，确保程序稳定运行：
